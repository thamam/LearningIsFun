## Key Features (Planned)

### Voice-First Interaction
- Built-in browser speech recognition (speech-to-text)
- Text-to-speech for questions and stories
- No additional software needed

### Pedagogical Foundation
- TPRS-inspired storytelling approach
- Oral language development before literacy
- Scaffolded progression (easy ‚Üí medium ‚Üí hard)
- High-frequency vocabulary for 8-year-olds

### Technology Stack
- Pure HTML/CSS/JavaScript (no framework)
- Web Speech API for voice interaction
- LocalStorage for progress persistence
- Responsive design (desktop, tablet, mobile)

### Four Practice Modules
1. **Listen & Respond** - Comprehension and vocabulary
2. **Speak & Practice** - Pronunciation and fluency
3. **Read & Match** - Sight words and phonics
4. **Write & Create** - Letter tracing and word copying

## Target Outcomes

**Month 1**: 25 words mastered, 70%+ listening accuracy
**Month 3**: 50 words, 10+ spoken sentences, 7-day streak
**Month 6**: 100 words, simple conversations, basic reading
